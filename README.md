## Веб-мессендер

Данный веб-мессенджер позволяет общаться в интернете.

## Установка

- NodeJS - 20.9.0

- `npm run start` - запуск для разработки
- `npm run build` - сборка проекта
- `npm run server` - запуск express сервера
- `npm run type-check` - проверка типов
